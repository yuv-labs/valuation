## 1. 회사 개요

Teradata Corporation은 1979년 Caltech과 Citibank 연구진 협업으로 설립된 엔터프라이즈 데이터 분석 및 클라우드 데이터 웨어하우스 소프트웨어 기업. 2007년 NCR에서 분사하여 독립 상장. 본사는 캘리포니아 샌디에이고.

**사업 모델:** 구독 기반 데이터 웨어하우싱/분석 플랫폼. 온프레미스에서 클라우드(VantageCloud)로 전환 중. 매출의 87%가 Recurring Revenue(구독 + 유지보수).

**제품 포트폴리오:**

| 제품군 | 주요 기능 | 비고 |
|--------|-----------|------|
| VantageCloud Lake | 클라우드 네이티브 데이터 레이크하우스 | AWS, Azure, Google Cloud 지원 |
| VantageCloud Enterprise | 하이브리드 클라우드 데이터 웨어하우스 | 기존 온프레미스 고객 마이그레이션 경로 |
| ClearScape Analytics | AI/ML 분석 엔진 | 내장 분석 함수, LLM 통합, RAG 지원 |
| Teradata Consulting | 구현, 마이그레이션, 최적화 서비스 | 매출 비중 감소 중 |

**핵심 성장 동력:**

1. **클라우드 마이그레이션:** 온프레미스 고객을 VantageCloud로 전환. Cloud ARR $701M(+15% YoY)으로 전체 ARR의 46%.
2. **AI/GenAI 통합:** NVIDIA NeMo/NIM 통합, 벡터 스토어 강화, Agentic workflow 지원. ClearScape Analytics로 AI 워크로드 플랫폼화.
3. **하이브리드/멀티클라우드 전략:** 78%의 기업이 하이브리드 모델로 전환 중(2026 전망) — Teradata의 하이브리드 포지셔닝이 강점.

**핵심 리스크:**

1. **총매출 역성장:** FY2025 매출 $1.66B(-5% YoY). 온프레미스 매출 감소가 클라우드 성장을 상쇄.
2. **클라우드 네이티브 경쟁사 압도적 성장:** Snowflake(~$3.4B ARR, 20% 시장점유율), Databricks(~$3.7B ARR, 57% 성장).
3. **총 ARR 정체:** $1.52B(+3% YoY) — 온프레미스 ARR 감소($821M, ~-10% YoY)가 클라우드 ARR 성장을 거의 상쇄.

### 2-5. PBR 역산

PBR = ((1+ROE)/(1+r))^N 공식 적용. 자기자본이 자사주 매입으로 인위적으로 낮아 PBR 10.8x이나, ROE도 비정상적으로 높음(NI $130M / Equity $230M = 56.5%).

| 할인율 r | 시장이 가정하는 ROE 유지기간 N |
|---------|-----|
| 10% | 5.6년 |
| 12% | 6.7년 |

시장은 ROE 56.5%가 5~7년간 유지될 것으로 가정. 그러나 ROE가 자사주 매입으로 인한 회계적 효과이므로, **ROIC로 대체 평가하는 것이 적절**. FCF/Invested Capital 기준 ROIC ≈ 170%라는 수치는 투하자본이 극소화된 소프트웨어 특성. PBR 역산은 TDC에 대해서는 유의미한 지표가 아님.

## 3. 경제적 해자 평가

### 프랜차이즈 3조건 (버핏)

| 조건 | 판정 | 근거 |
|------|------|------|
| 1. 소비자가 필요로 하거나 욕망한다 | **YES** | 대기업 데이터 분석은 경영 필수 인프라. Fortune 100 중 다수가 Teradata 고객. |
| 2. 소비자 마음속에 대체재가 없다 | **NO** | Snowflake, Databricks, AWS Redshift, Google BigQuery, Azure Synapse 등 다수 대체재. |
| 3. 가격 규제가 없다 | **YES** | 규제 없음. 가격은 경쟁과 계약 조건에 의해 결정. |

프랜차이즈 3조건 중 2/3 충족. **"필요하지만 대체재가 풍부한" 사업** — 전환비용과 기존 고객 관계가 핵심 차별 요인.

### See's Candies 테스트

- **ROIC:** 자기자본이 극히 낮아($230M) 산출하면 수치가 왜곡됨. FCF $285M / (Equity $230M + Net Debt -$62M) ≈ **170%** — 그러나 이는 자사주 매입으로 자기자본이 인위적으로 낮아진 결과.
- **CAPEX/Revenue 1.2%** → 극히 낮음. 소프트웨어 사업의 장점.
- **FCF($285M) > NI($130M)** → 현금 전환율 매우 높음. D&A와 주식보상이 크고, CAPEX는 적음.

### 해자 종류별 평가

| 해자 유형 | 등급 | 근거 |
|-----------|------|------|
| 브랜드 | **MODERATE** | 40년+ 엔터프라이즈 데이터 웨어하우싱 레거시. Fortune 500 기업에서 인지도 높음. 그러나 "혁신"보다 "레거시"로 인식되는 리스크. |
| 네트워크 효과 | **NONE** | 단일 기업 내 도구. 외부 네트워크 효과 없음. |
| 전환비용 | **HIGH** | **핵심 해자.** 대기업의 데이터 웨어하우스 교체는 수년~수천만 달러 프로젝트. SQL 쿼리, ETL 파이프라인, 리포트, 권한 체계 모두 재구축 필요. 마이그레이션 실패 리스크 높음. |
| 원가우위 | **LOW** | 클라우드 네이티브 경쟁사(Snowflake, Databricks) 대비 원가 구조에서 불리. 온프레미스 하드웨어 부문 잔존. |
| 규모의 경제 | **MODERATE** | 대규모 병렬처리(MPP) 기술에 40년 투자 축적. 그러나 클라우드에서는 하이퍼스케일러의 규모에 밀림. |

### 핵심 해자: 전환비용(Switching Cost) 심층 분석

Teradata의 핵심 해자는 전환비용. 대기업 고객이 수십 년간 축적한 SQL 쿼리, 분석 파이프라인, 리포트 체계를 Snowflake/Databricks로 옮기려면:

1. **기술적 비용:** Teradata SQL 방언 → ANSI SQL 변환, 저장 프로시저 재작성, 성능 튜닝
2. **조직적 비용:** DBA 재교육, 수천 명의 분석가 도구 전환, 다운타임 리스크
3. **금전적 비용:** 컨설팅 비용 $10M~$50M+ (대기업 기준), 병행 운영 비용
4. **리스크 비용:** 마이그레이션 실패 시 비즈니스 인텔리전스 중단 → 경영 의사결정 마비

**그러나 이 해자는 "방어적"이지 "공격적"이지 않음.** 기존 고객을 유지할 수는 있지만, 신규 고객 획득에서는 Snowflake/Databricks에 밀림. Greenfield(새 프로젝트)에서 Teradata를 선택하는 기업은 거의 없음.

### 슈퍼스타 CEO 리스크

버핏: "슈퍼스타가 있어야만 훌륭한 성과를 낼 수 있는 사업이라면 훌륭한 사업이라고 평가할 수 없습니다."

Steve McMillan CEO가 클라우드 전환을 주도하고 있으나, Teradata의 핵심 가치(전환비용)는 CEO 개인이 아닌 40년간 축적된 고객 기반에서 나옴. CTO Stephen Brobst는 25년+ 재직으로 기술적 연속성 확보. **슈퍼스타 의존도: 낮음.**

### 가치 창출형 성장 확인

버핏: "재투자가 많이 필요하지 않은 채로 성장하는 기업이 진정으로 가치 있는 기업."

- CAPEX/매출 1.2% → 극히 낮음. 소프트웨어 사업의 본질적 장점.
- FCF $285M vs 매출 $1.66B → FCF Margin 17.1%.
- 문제는 **성장 자체가 없다는 것**. 총매출 -5% YoY. "재투자가 적은 성장"이 아니라 "재투자도 적고 성장도 없음."
- 버핏이 선호하지 않는 사업 중 "자금을 쏟아붓다 보면 언젠가 대량의 현금흐름을 회수해줄 사업"도 아님 — 이미 충분한 FCF를 창출하고 있으나 성장 엔진이 약함.

## 4. 경쟁 포지셔닝

### 시장 환경

클라우드 데이터 웨어하우스/레이크하우스 시장은 약 $30B+(2025 추정)으로 급성장 중. Teradata는 **레거시 온프레미스 데이터 웨어하우스의 클라우드 전환** 시장에서 경쟁.

| 경쟁사 | ARR/매출 | 성장률 | 특징 |
|--------|----------|--------|------|
| **Snowflake** | ~$3.4B ARR | ~25% | 클라우드 네이티브 DW 1위. 시장점유율 ~20%. |
| **Databricks** | ~$3.7B ARR | ~57% | 레이크하우스 선두. AI/ML에 강점. 비상장(밸류에이션 $62B). |
| **AWS Redshift** | 비공개 | — | AWS 생태계 내 번들. 가격 경쟁력. |
| **Google BigQuery** | 비공개 | — | 서버리스, AI 통합. Google Cloud 번들. |
| **Azure Synapse** | 비공개 | — | Microsoft 생태계 통합. |
| **Teradata** | $1.52B ARR | +3% | 하이브리드/멀티클라우드. 대기업 레거시 기반. |

**DB-Engines 랭킹:** Snowflake 6위, Teradata 22위 (2025.01 기준). 이는 기술 커뮤니티에서의 관심도와 채택률 격차를 반영.

### Teradata의 경쟁 전략

1. **하이브리드 클라우드:** 온프레미스와 클라우드를 단일 쿼리로 연결. 완전한 클라우드 전환이 어려운 대기업에 소구.
2. **멀티클라우드:** AWS, Azure, Google Cloud 모두 지원. 벤더 락인 회피.
3. **비용 효율성:** 대규모 데이터셋에서 Snowflake 대비 TCO(Total Cost of Ownership) 우위 주장.
4. **엔터프라이즈 SLA:** 대기업이 요구하는 고가용성, 보안, 성능 보장.

### 경쟁 상의 구조적 약점

- **인재 확보:** 신진 데이터 엔지니어는 Snowflake/Databricks/dbt를 배움. Teradata 스킬셋은 줄어드는 인력 풀.
- **생태계:** Snowflake/Databricks는 수천 개의 파트너/통합. Teradata 생태계는 상대적으로 제한적.
- **인식:** Gartner Peer Insights에서 Teradata는 경쟁사 대비 가치(value), 비용(cost), 지원(support) 모두 낮은 평가.

## 6. Fisher 15 기준 + Lynch 분류

### Lynch 6분류: **턴어라운드(Turnaround)**

총매출이 4년 연속 정체~역성장($1.84B→$1.66B). 클라우드 전환이 성공하면 성장 복귀 가능하나, 현재는 전형적인 턴어라운드 스토리. 온프레미스 감소를 클라우드 성장이 상쇄하는 "교차점(crossover)"이 언제 오는지가 핵심 질문.

### PEG 비율

- Trailing PER 19.8x / 총매출 성장률 -5% = **음수** (의미 없음)
- Forward PER 10.2x / Non-GAAP EPS 성장률 ~3-5% = **PEG 2.0~3.4**
- Lynch 기준: **비싸다** — 그러나 PEG는 성장주에 적합한 지표이고, 턴어라운드에는 P/FCF가 더 적절.

## 10. 교차검증

### 10-1. Reverse DCF

### 10-2. P/FCF 비교 분석

| 기업 | P/FCF | 매출 성장률 | FCF Margin | 비고 |
|------|-------|------------|------------|------|

### 핵심 리스크 심층 분석

#### 11-1. 온프레미스 이탈 가속 — 실제 심각도 ★★★★★

**가장 심각한 리스크.** On-Prem ARR이 $1.04B(2023) → $865M(2024) → $821M(2025)로 매년 ~$50~175M씩 감소. 이 감소분이 Cloud ARR 증가분($80~90M/yr)을 초과하면 Total ARR이 역성장.

문제는 온프레미스 이탈의 두 가지 경로:

1. **Teradata Cloud로 마이그레이션** → ARR 유지(긍정적)
2. **Snowflake/Databricks로 이탈** → ARR 영구 손실(부정적)

Cloud Net Expansion Rate 108%는 기존 클라우드 고객이 소비를 늘리고 있음을 보여주나, 117%(Q4 2024) → 108%(Q4 2025) 하락 추세는 우려.

#### 11-2. 구조적 경쟁 열위 — 실제 심각도 ★★★★☆

Snowflake($3.4B ARR, +25%)와 Databricks($3.7B ARR, +57%)는 Teradata의 2~3배 규모이면서 5~20배 빠르게 성장. 이 격차는 시간이 갈수록 벌어짐.

Teradata의 유일한 구조적 강점은 **기존 대기업 고객 기반**이나, 이는 감소하는 자산(depleting asset). 새로운 대기업 고객을 획득하지 못하면 TAM이 지속 축소.

#### 11-3. CEO 보상 구조 — 실제 심각도 ★★☆☆☆

CEO 총보상 $17.4M은 시가총액 $2.5B 기업에 비해 상당히 높음(시가총액의 0.7%). 역성장 기업의 경영진에게 이 수준의 보상이 정당화되는지 의문. 다만 이것만으로 투자 판단을 바꿀 수준은 아님.

### 과신 점검

"왜 이 가격인가?"

시장의 논리:

1. **총매출 4년 연속 역성장:** $1.84B(2020) → $1.66B(2025). 구조적 쇠퇴 우려.
2. **클라우드 네이티브 경쟁사에 밀림:** Snowflake/Databricks가 시장을 장악하는 가운데 Teradata는 레거시 전환에 의존.
3. **Cloud Net Expansion Rate 하락:** 117% → 108%. 클라우드 내에서도 성장 모멘텀 약화.
4. **높은 Short Interest(13.1%):** 시장 참여자들의 활발한 하방 베팅.
5. **Total ARR 겨우 +3%:** 소프트웨어 기업에서 +3%는 사실상 정체.

**과신 경고 신호 체크:**

- P/FCF 8.9x는 "시장이 틀렸다"는 주장을 뒷받침하기에 매력적. 그러나 이렇게 낮은 배수는 **FCF가 미래에 감소할 것이라는 시장의 예측**을 반영.
- Reverse DCF가 가정하는 FCF 성장률(0%)은 비관적이나 비현실적이지 않음 — 온프레미스 이탈이 가속되면 실현 가능.
- **"mini Oracle" 비유에 대한 경고:** Oracle은 데이터베이스 외에 ERP, Cloud Infra 등 다각화 옵션이 있었음. Teradata는 데이터 웨어하우스 단일 사업으로 다각화 여지가 제한적.
- Short Interest 13.1%는 시장이 단순히 "관심 없다"가 아니라 적극적으로 하락에 베팅하고 있음을 시사.

**확신도: 낮음~중간** — 밸류에이션은 매력적이나, 구조적 경쟁 열위와 매출 역성장이 "가치 함정(value trap)"일 가능성을 배제할 수 없음.

### 2분 테스트 (Lynch)

"Teradata는 40년 역사의 엔터프라이즈 데이터 웨어하우스 회사다. 대기업들이 Teradata에서 Snowflake/Databricks로 옮기는 것은 매우 어렵고 비싸서(전환비용), 기존 고객은 잘 떠나지 않는다. 그러나 새 고객은 거의 들어오지 않고, 총매출은 4년째 줄고 있다($1.84B→$1.66B). 회사는 클라우드(VantageCloud)로 전환 중이고, 클라우드 매출은 +15% 성장하지만 온프레미스 감소를 겨우 상쇄하는 수준이다. FCF는 연 $285M으로 견실하고 P/FCF 8.9x로 싸다. 그러나 Snowflake($3.4B ARR, +25%)와 Databricks($3.7B ARR, +57%)에 비하면 규모도 작고 성장도 느려서, '싼 데는 이유가 있다'는 시장의 판단이 틀렸다는 확신이 없다."

**결론: 관망(Watchlist). 밸류에이션은 매력적이나, 가치 함정(value trap) 리스크가 높고 구조적 경쟁 열위에 대한 확신이 부족. 모니터링 포인트: (1) Total ARR 성장률이 +5% 이상으로 가속하는지, (2) Cloud Net Expansion Rate가 반등하는지, (3) 클라우드 ARR이 온프레미스 ARR을 넘어서는 교차점(2026H2 예상). 이 세 조건 중 2개 이상 확인되면 재평가.**
