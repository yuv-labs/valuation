## 1. 회사 개요

Fortinet은 1999년 Ken Xie(CEO/Chairman)와 Michael Xie(President/CTO) 형제가 공동 설립한 사이버보안 기업. 네트워킹과 보안의 융합(convergence)을 핵심 전략으로 추진. 본사: Sunnyvale, CA. 직원 수: 15,109명(FY2025 말).

**사업 모델:** 자체 설계 ASIC(Application-Specific Integrated Circuit) 기반 FortiGate NGFW(Next-Generation Firewall) 하드웨어 판매 + FortiGuard 보안 구독 + FortiCare 기술지원의 "razor-and-blades" 모델. 어플라이언스 판매 후 수년간 서비스 매출을 수반하는 구조.

**핵심 제품/플랫폼:**

| 제품/플랫폼 | 설명 | 포지셔닝 |
|------------|------|----------|
| FortiGate NGFW | 자체 ASIC 칩 기반 차세대 방화벽. 유닛 기준 시장점유율 55% | 핵심 캐시카우 |
| FortiOS | 50개 이상 제품을 관통하는 단일 OS | 통합 플랫폼의 기반 |
| Fortinet Security Fabric | 네트워크/엔드포인트/클라우드/AI 통합 포트폴리오 | 플랫폼 lock-in |
| FortiSASE (Unified SASE) | SD-WAN + SSE(Secure Service Edge) 통합 | 고성장 영역 |
| FortiGuard AI Security Services | 위협 인텔리전스 + ML 기반 실시간 보안 | 구독 매출 드라이버 |

**유통 구조:** 2-tier 모델(Distributor → Reseller/MSP). 상위 3개 유통사가 매출의 55%. 최종 고객 50만+ 이상.

**핵심 성장 동력:**

1. **FortiGate Refresh Cycle:** 유닛 점유율 55%의 설치 베이스에서 3-5년 주기 교체 수요 발생. FY2024 일시 둔화 후 FY2025 product revenue +16% 반등. 하드웨어 교체는 이후 수년간 FortiGuard/FortiCare 서비스 매출의 선행지표.
2. **Unified SASE 확장:** SD-WAN + SSE 통합 차세대 아키텍처. Q4'25 Unified SASE billings +40% YoY. Gartner Peer Insights Customers' Choice for SSE 3년 연속.
3. **AI Factory/OT Security:** NVIDIA BlueField-3 DPU 통합, FortiGate Rugged(OT 보안). 산업용 IoT/ICS 환경에서 빠르게 성장하는 니치 시장.

**핵심 리스크:**

1. **방화벽 시장 구조적 전환:** 클라우드 네이티브 보안(Zscaler, Cloudflare)과 SASE 전환이 전통 하드웨어 방화벽 시장을 구조적으로 축소시킬 가능성. 매출의 33%가 product revenue.
2. **경쟁 심화:** Palo Alto Networks($127B MCap, "platformization"), CrowdStrike($96B, endpoint/identity), Zscaler(cloud-native SSE)와 직접 경쟁. 사이버보안은 경쟁이 매우 치열한 시장.
3. **유통 집중도:** 상위 3개 유통사가 매출의 55%. 유통사 재고 조정이 분기 실적 변동성을 키울 수 있음.

**현재 주가 상황:** 52주 고점($115) 대비 ~27% 하락. PER 33.6x. 애널리스트 컨센서스: Hold, 목표가 $87 (+4%).

### 2-6. PBR 역산

PBR = ((1+ROE)/(1+r))^N 공식은 ROE 135.7%라는 극단적 수치와 PBR 50.3x의 조합으로 의미 있는 해석이 어려움. ROIC 기반 분석이 더 적절.

r=10%에서 N = 5.1년, r=12%에서 N = 5.3년. 자본 구조가 왜곡되어 있으므로 DCF 분석에 의존.

## 4. Fisher 15 기준 + Lynch 분류

### Lynch 6분류: **대형 우량주 (Stalwart)**

매출 +14%, EPS +8%. 고성장주(30%+)보다는 안정적이나 저성장주(5% 미만)는 아님. 시가총액 $62B의 대형주로서 안정적 이중자릿수 성장. 사이버보안 시장이 구조적으로 성장하므로 경기순환주는 아님.

### PEG 비율

- PER(GAAP) 33.6x / 예상 OE 성장률 ~12% = **PEG 2.80**
- PER(Non-GAAP forward) 28x / 예상 EPS 성장률 ~12% = **PEG 2.33**
- Lynch 기준: PEG > 2이면 "매우 불리". **현재 주가는 성장 대비 프리미엄이 높음.**

#### 8-1. 방화벽 → 클라우드 네이티브 전환 — 실제 심각도 ★★★☆☆

하드웨어 방화벽이 "레거시"가 되는 시나리오. Zscaler, Cloudflare 같은 cloud-native 보안이 on-premise 방화벽을 대체할 가능성. 그러나:

- 위험도는 "갑작스러운 붕괴"보다 "점진적 전환"에 가까움

#### 8-2. 경쟁 심화 (Palo Alto Platformization) — 실제 심각도 ★★★☆☆

Palo Alto Networks가 보안 플랫폼 번들 할인 전략으로 고객 유치. Fortinet의 대응: Security Fabric(50개 제품 통합) + ASIC 기반 TCO 우위. 양사 모두 "플랫폼" 전략을 추구하지만 접근 방식이 다름(PANW는 SW 중심, FTNT는 HW+SW 수직통합). 단기적으로 가격 압박 가능, 구조적 파괴 가능성은 낮음.

#### 8-3. 형제 경영 리스크 — 실제 심각도 ★★☆☆☆

Ken Xie(CEO)와 Michael Xie(CTO)의 형제 경영은 25년간 성공적이었으나, 경영진 depth 우려와 승계 리스크가 존재. 두 사람 모두 60대로 추정되며, 후임 CEO 승계 계획이 불투명. ASIC 설계 역량이 특정 개인에 귀속되지 않는다면 구조적 위험은 제한적.
