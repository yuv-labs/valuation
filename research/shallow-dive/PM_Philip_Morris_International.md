## Phase 2. 2분 테스트 + 사업모델

### Lynch 2분 피치

PM은 미국 외 시장에서 **Marlboro** 궐련담배를 판매하며, 감소하는 combustible 기반을 3개 스모크프리 플랫폼으로 전환 중: **IQOS** (가열식 담배, 일본·EU 지배), **ZYN** (니코틴 파우치, 현재 미국 카테고리 value share ~67%, 카테고리 CAGR ~30%), **VEEV** (전자식). 니코틴 중독이 반복구매를 유발하고, 규제가 incumbent를 보호하며, PM은 IQOS와 ZYN 20종에 대해 모두 **FDA 마케팅 승인을 보유한 유일한 글로벌 플레이어** (2025-01). 투자 논리는 전형적 "현금소 + kicker": 궐련으로 현금 짜내고, Zyn·IQOS를 peers보다 빠르게 스케일 업.

### 10-A-2. Risk / Opportunity 매핑

### 10-A-3. 계산

**Bull**:

- 10yr 매출 = 40.6 × 1.08^10 = $87.65B
- 10yr NI = 87.65 × 0.40 × 0.78 = $27.35B
- 10yr 주식수 = 1,558 × 0.985^10 = 1,340M
- 10yr EPS = $20.41
- 10yr 주가 = 20.41 × 22 = $449
- 누적 배당 (초기 $5.88, 9%/yr 10년 합) ≈ $89
- 10yr 주당 총가치 = 449 + 89 = **$538**
- PV = 538 / 1.10^10 = **$207**

**Base**:

- 10yr 매출 = 40.6 × 1.06^10 = $72.71B
- 10yr NI = 72.71 × 0.37 × 0.78 = $20.99B
- 10yr 주식수 = 1,558 × 0.99^10 = 1,409M
- 10yr EPS = $14.90
- 10yr 주가 = 14.90 × 18 = $268
- 누적 배당 (8%/yr) ≈ $85
- 10yr 주당 총가치 = **$353**
- PV = 353 / 2.594 = **$136**

**Bear**:

- 10yr 매출 = 40.6 × 1.03^10 = $54.56B
- 10yr NI = 54.56 × 0.33 × 0.78 = $14.04B
- 10yr 주식수 = 1,558M (flat)
- 10yr EPS = $9.01
- 10yr 주가 = 9.01 × 14 = $126
- 누적 배당 (4%/yr) ≈ $71
- 10yr 주당 총가치 = **$197**
- PV = 197 / 2.594 = **$76**

### 10-A-4. 결과 + 확률가중

| 시나리오 | 10yr 총가치 | PV | 현재가 대비 | 실현 CAGR |

| Base (50%) | $353 | $136 | -26% | **6.8%** |
| Bear (20%) | $197 | $76 | -58% | **0.8%** |

**확률가중 PV**: 0.3×207 + 0.5×136 + 0.2×76 = **$145**
**현재가/확률가중 PV**: 182.81/145 = **126%** (현재가 26% 고평가)
**확률가중 CAGR**: 0.3(11.4) + 0.5(6.8) + 0.2(0.8) = **6.98%**

**확률 근거**: Bull 30% (ZYN 규제 해소 + IQOS US 성공 + margin 확장 동시 실현은 3연타 필요), Base 50% (경영진 가이던스 중간값 이행), Bear 20% (규제 tail 실현 가능성 기본치).

### 10-A-5. Trigger 신호

- **Bear PV $76 << 현재가 $182.81**: 극단 저가 신호 **미해당** (Bear도 현재가 대비 -58%)
- **현재가 $182.81 < Bull PV $207**: 과신 경고 **미해당** (Bull이 현재가 상회)
- **Base PV $136 < 현재가**: Base 시나리오 이상 실현 필요 — 경영진 가이던스 상단(EPS CAGR 11%) 10년 지속을 price-in 중. **Deep 진입 우선순위는 "경영진 가이던스 이행 증명력"에 가장 민감**.

### 10-A-6. Deep Section 6 연계 signal

- 확률가중 CAGR 6.98% < r 10% → **현재가는 프리미엄**. Shallow 기준 "매수 가능"은 Bull 시나리오 가정치 필요.
- Deep에서 정밀화 필수 축: (a) Zyn US 볼륨 +37% 지속가능성 (b) IQOS 미국 TAM 진입률 (c) Deleveraging 속도 + buyback 재개 타이밍
- Reverse DCF (Deep 수행): 현재가 $182.81을 r=10%으로 정당화하려면 10yr EPS 약 $15~17 (Base와 Bull 사이) + 터미널 PER 18~20x 필요.
