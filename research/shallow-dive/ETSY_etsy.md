## Phase 2. 2분 테스트 + 사업모델

### Lynch 2분 피치

Etsy는 전 세계 핸드메이드/빈티지/유니크 상품 셀러 560만 명과 바이어 8,650만 명을 연결하는 **양면 마켓플레이스**로, 거래액(GMS)의 약 24.5%를 수수료로 수취한다. Amazon과 대형 리테일러가 제공하지 못하는 "유일하고 특별한 물건"이라는 포지셔닝이 핵심이다. 연간 FCF $680M을 꾸준히 창출하면서 GMS 성장이 정체된 구간이라 현금환원(buyback)에 집중하고 있어, 만약 GMS 재성장에 성공하면 현재 FCF yield 9%대의 가격이 재평가될 수 있다.

### PBR 공식 역산

자기자본 음수로 PBR 공식 직접 적용 불가. ROIC 기반 대체 사용:

```text
r = (1+ROIC) / (EV/IC)^(1/N) - 1
```

- ROIC (FY2024 기준): 31.5%
- EV: ~$8.6B, IC (FY2024): $941M
- EV/IC: 9.2x
- N = 10: r = 1.315 / 9.2^(1/10) - 1 = 1.315 / 1.248 - 1 = **5.4%**
- N = 7: r = 1.315 / 9.2^(1/7) - 1 = 1.315 / 1.371 - 1 = **-4.1%**

시장이 가정하는 초과수익 지속 기간: ROIC 31.5% 유지 시 N~10년에서 r=5.4%로 10% 미달. 시장은 **ROIC가 현 수준 유지 불가**하거나 **지속 기간이 짧다**고 본다는 의미. 또는 IC 계산이 buyback으로 왜곡.

**참고**: Equity 음수 기업이라 전통 PBR 역산이 의미 없음. FCF yield 9.4%가 더 직관적인 밸류에이션 앵커.

### 10-A-3. 계산

기준: FY2025 매출 $2,884M, 주식수 124.1M, 현재가 $57.92, 세율 22%

**Bull**:

- 10년 후 매출: $2,884M x (1.07)^10 = $5,674M
- 10년 후 NI: $5,674M x 16% x (1-0.22) = $708M
- 10년 후 주식수: 124.1M x (0.96)^10 = 82.3M
- 10년 후 EPS: $708M / 82.3M = $8.60
- 10년 후 가격: $8.60 x 18 = $154.8
- PV: $154.8 / (1.10)^10 = **$59.7**

**Base**:

- 10년 후 매출: $2,884M x (1.04)^10 = $4,270M
- 10년 후 NI: $4,270M x 13% x (1-0.22) = $433M
- 10년 후 주식수: 124.1M x (0.97)^10 = 91.6M
- 10년 후 EPS: $433M / 91.6M = $4.73
- 10년 후 가격: $4.73 x 14 = $66.2
- PV: $66.2 / (1.10)^10 = **$25.5**

**Bear**:

- 10년 후 매출: $2,884M x (1.01)^10 = $3,186M
- 10년 후 NI: $3,186M x 9% x (1-0.22) = $224M
- 10년 후 주식수: 124.1M x (0.99)^10 = 112.9M
- 10년 후 EPS: $224M / 112.9M = $1.98
- 10년 후 가격: $1.98 x 10 = $19.8
- PV: $19.8 / (1.10)^10 = **$7.6**
