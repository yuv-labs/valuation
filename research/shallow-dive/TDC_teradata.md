### PBR 공식 역산

PBR은 equity 왜곡으로 사용 불가. ROIC 기반 EV/IC 대체 사용.

```text
EV/IC = ((1+ROIC)/(1+r))^N
```

현재: EV/IC 약 6.9x, ROIC 43%, r = 10%

- r=10%: 시장이 가정하는 N = **9.0년** (ROIC 43%가 10년 가까이 유지된다는 암묵 가정)
- r=12%: N = **9.7년**

해석: 현재 가격은 Teradata가 약 9~10년간 높은 ROIC(43%)를 유지한다고 가정. 매출 감소 추세 감안 시 이 가정은 **낙관적**일 수 있음 -- IC가 계속 축소되면서 ROIC가 높아 보이는 효과.

Teradata는 **엔터프라이즈 데이터 웨어하우스 레거시 시장**에서 높은 전환비용에 기반한 고객 유지력을 보유하고 있다. 클라우드 전환기에 매출은 연 2~5% 감소하지만, ARR 모델 전환과 비용 효율화로 마진과 FCF를 방어하고 있다. AI 워크로드 확대와 하이브리드 클라우드 전략이 매출 안정화를 이끌어낼 수 있어 보이며, 적극적 buyback이 주당 가치를 지탱한다. 다만 **Snowflake/Databricks와의 경쟁 심화가 구조적 역풍**이며, 매출 반등 시점과 크기가 불확실하다.

### 10-A-1. 시나리오 가정 매트릭스

| 변수 | Bull | Base | Bear |
| --- | --- | --- | --- |
| 매출 CAGR (10년) | +2% | -1% | -5% |
| 터미널 EBIT 마진 | 15% | 13% | 8% |
| 주식수 연간 변화 | -3% | -2% | -1% |
| 터미널 PER | 18x | 14x | 8x |

가정 근거:

- **Bull**: 클라우드 ARR 성장 + AI 워크로드가 온프레미스 감소를 상쇄하고 매출 반등. 마진 확대 지속. 적극 buyback. Snowflake/Databricks와 공존 성공.
- **Base**: 매출 완만 감소 지속(-1%/yr). 마진 소폭 개선. Buyback 지속하나 규모 축소.
- **Bear**: 대형 고객 이탈 가속, 클라우드 경쟁에서 밀림. 마진 압축. Buyback 여력 감소.

### 10-A-3. 계산

```text
현재 매출: $1,663M | 현재 주식수: 94.5M | 세율: 22% | r: 10%

Bull:
  10yr 매출 = 1,663 x 1.02^10 = $2,027M
  NI = 2,027 x 0.15 x 0.78 = $237M
  주식수 = 94.5 x 0.97^10 = 69.7M
  EPS = $3.40 -> 10yr Price = $3.40 x 18 = $61.24
  PV = $61.24 / 1.10^10 = $23.61

Base:
  10yr 매출 = 1,663 x 0.99^10 = $1,504M
  NI = 1,504 x 0.13 x 0.78 = $153M
  주식수 = 94.5 x 0.98^10 = 77.2M
  EPS = $1.97 -> 10yr Price = $1.97 x 14 = $27.64
  PV = $27.64 / 1.10^10 = $10.66

Bear:
  10yr 매출 = 1,663 x 0.95^10 = $996M
  NI = 996 x 0.08 x 0.78 = $62M
  주식수 = 94.5 x 0.99^10 = 85.5M
  EPS = $0.73 -> 10yr Price = $0.73 x 8 = $5.81
  PV = $5.81 / 1.10^10 = $2.24
```

### 10-A-4. 결과

| 시나리오 | PV | 현재가 대비 | 실현 CAGR |
| --- | --- | --- | --- |
| Bull (20%) | $23.61 | -9% | +8.9% |
| Base (55%) | $10.66 | -59% | +0.6% |
| Bear (25%) | $2.24 | -91% | -13.9% |

**확률가중 PV**: 0.20 x $23.61 + 0.55 x $10.66 + 0.25 x $2.24 = **$11.14**
**현재가($26.00) 대비**: **-57%**

확률 배분 근거: 매출 역성장 추세 5년 지속 + 산업 구조적 전환(클라우드 네이티브 우위)을 감안하여 Bear에 25%, Base에 55% 비중. Bull은 AI 전환 성공 시나리오로 20%.

### 10-C. Trigger 신호

- **현재가($26) > Bull PV($23.61)**: 가장 낙관적 시나리오의 PV보다 현재가가 높음. **과신 경고** 영역.
- 자문: "내가 성장 동력(AI, 하이브리드 클라우드)을 과소평가하고 있는가?" -- 가능하나, 5년간 매출 역성장이 지속되는 현실이 가정을 뒷받침.
- Base/Bear PV($10.66/$2.24) 모두 현재가 대비 크게 하회: 현재 가격에는 **상당한 매출 반등 기대가 반영**되어 있음.
- FCF Yield 11.6%가 높아 보이지만, 매출 감소가 지속되면 FCF도 축소될 수밖에 없음.

### 확신도: 낮음

**판정 근거**: Teradata는 높은 전환비용에 기반한 고객 유지력과 견조한 FCF 창출이 있으나, **5년 연속 매출 역성장**, **클라우드 네이티브 경쟁자 대비 구조적 열위**, **Bull 시나리오에서도 현재가 정당화 어려움**이 결합되어 Deep Dive에 1~2주를 투자할 가치가 확인되지 않는다. 현재 가격($26)은 AI 전환 성공과 매출 반등에 대한 기대를 상당히 반영하고 있으며, 이 기대가 실현되지 않을 경우 하방 리스크가 크다.

### 과신 점검

> "왜 싸가?"

Teradata 주가는 2022년 고점 $50에서 -48% 하락. 시장은 (1) 매출 역성장 추세, (2) 클라우드 네이티브 경쟁자 대비 구조적 약점, (3) AI 전략의 불확실한 성과를 반영하고 있다. 이 할인은 합리적으로 보인다.

> "왜 비싼가?"

현재 PER 18.9x, FCF Yield 11.6%는 절대값으로 비싸 보이지 않으나, **매출 감소 기업**에 적용하면 다른 그림이 나온다. DCF 분석에서 Bull PV도 현재가를 하회하는 점이 이를 뒷받침한다.

> "왜 이 트랙인가?"

매출 역성장, 낮은 Capex, 적극적 buyback, 성숙 시장의 현금 회수 패턴 -- 전형적 버핏형 특성. 다만 경영진의 AI 재투자 내러티브가 혼합 요소.

### 다음 행동

- **Parking Lot**: 분기 리뷰 대상. AI 전략의 실질 성과(매출 반등, ARR 가속)가 확인되면 재진입 검토.
- 재진입 조건: (1) Total ARR 성장률 5%+ 연속 2분기, (2) 매출 역성장 폭 축소(-1% 이내), (3) 주가 $18~20 수준 도달 시 밸류에이션 재검토.
