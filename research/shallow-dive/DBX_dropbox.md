## Phase 2. 2분 테스트 + 사업모델

### Lynch 2분 피치

Dropbox는 개인과 기업에게 클라우드 파일 동기화·저장·공유 서비스를 구독(subscription) 방식으로 제공하여 돈을 번다. 7억 등록 사용자 중 1,808만 유료 사용자(ARPU $139.68)가 연간 $2.5B 매출을 만든다. 매출 성장은 멈췄지만, GAAP 영업이익률 27%, FCF 마진 37%로 현금을 뽑아내고 있으며, 그 현금을 거의 전액 자사주 매입에 투입하여 주당가치를 끌어올리는 **현금 회수 머신** 구조다.

### PBR 공식 역산

> 자기자본이 음수(-$1.8B)이므로 원래 PBR 공식 적용 불가.
>
> **ROIC 기반 대체**: r = (1 + ROIC) / (EV/IC)^(1/N) - 1
>
> - ROIC = 27.5%, EV = $7.73B, IC(TA-Cash) = $1.95B, EV/IC = 3.96
> - r = (1.275) / (3.96)^(1/10) - 1 = **12.8%** (N=10, r=10% 기준)
>
> 시장은 현재 가격에서 약 12.8%의 장기 기대수익률을 암시. r=10% 대비 양호.
>
> r=12% 가정 시: 시장이 가정하는 N = log(1.275/1.12) / log(3.96^(1/N)) 역산 → 약 **10-11년** 초과수익 지속 가정.

### 10-A-3. 계산

**기준**: FY2025 매출 $2,521M, 현재 주식수 238.6M, 현재가 $24.27, r = 10%, 세율 22%

**Bull**:

- 10년 후 매출 = $2,521M x (1.02)^10 = $3,073M
- 10년 후 NI = $3,073M x 30% x 78% = $719M
- 10년 후 주식수 = 238.6M x (0.95)^10 = 142.9M
- 10년 후 EPS = $5.03, Price = $5.03 x 16 = $80.53
- PV = $80.53 / (1.10)^10 = **$31.05**

**Base**:

- 10년 후 매출 = $2,521M x (0.995)^10 = $2,398M
- 10년 후 NI = $2,398M x 27% x 78% = $505M
- 10년 후 주식수 = 238.6M x (0.96)^10 = 158.7M
- 10년 후 EPS = $3.18, Price = $3.18 x 13 = $41.38
- PV = $41.38 / (1.10)^10 = **$15.95**

**Bear**:

- 10년 후 매출 = $2,521M x (0.96)^10 = $1,676M
- 10년 후 NI = $1,676M x 20% x 78% = $261M
- 10년 후 주식수 = 238.6M x (0.98)^10 = 195.0M
- 10년 후 EPS = $1.34, Price = $1.34 x 9 = $12.07
- PV = $12.07 / (1.10)^10 = **$4.65**

### 10-A-4. 결과

| 시나리오 | PV | 현재가 대비 | 실현 CAGR |
| --- | --- | --- | --- |
| Bull | $31.05 | +27.9% | 12.7% |
| Base | $15.95 | -34.3% | 5.5% |
| Bear | $4.65 | -80.8% | -6.7% |

**확률가중 PV**: 0.20 x $31.05 + 0.55 x $15.95 + 0.25 x $4.65 = **$16.15**

확률 근거: 매출 역성장 추세 + Google/MS 경쟁 심화로 Base/Bear 합산 80% 부여. Dash 성공 확률 제한적으로 Bull 20%.

**현재가 대비**: -33.5%

### 10-C. Trigger 신호

**10-A (Buffett DCF) 트리거**:

- 현재가 $24.27 > 확률가중 PV $16.15 → **경고 구간**
- 그러나 Bull PV $31.05 > 현재가 → 극단 저가 아님
- Base PV $15.95가 현재가의 66% 수준 → **현재가에는 성장 기대 또는 buyback 프리미엄이 포함**

**자문**:

- "왜 싼가?" — 매출 역성장 -1.1%, 유료 사용자 감소, Google/MS 번들 위협. 시장은 Dropbox를 쇠퇴하는 레거시 SaaS로 가격 매김
- "왜 비싼가?" — FCF yield 16%는 극히 높음. 만약 FCF가 10년간 유지된다면, buyback만으로도 연 16% EPS 성장 가능. 이 간이 DCF 모델은 **누적 buyback의 복리 효과를 과소 반영**할 수 있음 (터미널 PER이 static이므로)
- **핵심 불확실성**: FCF $931M이 10년간 유지될 수 있는가? 매출 하락 속도에 달려 있음

### 확신도: **중간**

판정 근거: Dropbox는 4축 모두 버핏형 신호를 보이는 전형적 현금 회수 기업이다. FCF yield 16%와 공격적 buyback은 매력적이나, (a) 매출 역성장이 가속될 경우 FCF 지속성이 위협받고, (b) 부채 $2.8B + 자기자본 음수라는 재무 구조가 하방 리스크를 키우며, (c) 확률가중 PV $16.15가 현재가를 하회하여 Base/Bear 시나리오에서 손실 가능성이 높다.

Deep에서 확인해야 할 것:

- FCF 지속성: 매출 -3% 시나리오에서도 FCF $700M+ 유지 가능한가 (비용 구조 정밀 분석)
- 부채 상환 스케줄 vs buyback 지속 가능성 (convertible note 만기, 전환 조건)
- Dash AI의 실질 매출 기여 가능성 (TAM, conversion funnel)
- 유료 사용자 이탈률 정밀 분석 (cohort별, 지역별, 플랜별)

### 과신 점검

> "왜 싸가?" — 시장은 매출 역성장 + Google/MS 번들에 의한 구조적 침식을 가격에 반영. 이 판단이 옳다면 PER 13x도 비쌀 수 있다.
>
> "왜 비싼가?" — FCF yield 16%는 S&P 500 평균의 3배+. 만약 시장이 FCF 지속성을 과소평가하고 있다면 상당한 업사이드.
>
> "왜 이 트랙인가?" — 4축 모두 버핏형이며, 경영진 가이던스가 "FCF per share 성장"을 명시적 North Star로 선언. R&D가 높지만 Dash가 매출로 전환되기 전까지는 비용 구조 일부로 봐야 함.

Track Gate 4축 4:0 일치에도 확신도 "중간"으로 설정한 이유는 확률가중 PV가 현재가를 하회하기 때문이다. Deep에서 FCF 지속성이 확인되면 확신도 상향 가능.
