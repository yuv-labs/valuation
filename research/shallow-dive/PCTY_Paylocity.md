## 2. 2분 테스트 + 사업모델 (Phase 2)

**Lynch 2분 요약**: Paylocity는 **클라우드 기반 HCM·페이롤 SaaS**를 **미국 중소~중견기업(직원 10~5,000명 규모)**에 **월간 구독료 + 페이롤 처리 건당 수수료**로 판다. ADP·Paychex 같은 레거시 대비 모바일·AI 네이티브 UX로 차별화했고, 최근 Airbase 인수로 Finance(AP/spend management)까지 플랫폼을 확장 중. 왜 훌륭할 수 있는가 — **전환비용 높은 Workflow SaaS 특성 + 제품당 교차판매 레버리지 + $27~81B 시장에서 여전히 한 자릿수 점유율**.

| 항목 | 내용 |
| ------ | ------ |
| 제품/서비스 | Payroll, HR, Benefits, Talent Management, Workforce Mgmt, Spend Mgmt(Airbase), Premium AI assistant |
| 주요 고객 | 미국 SMB·중견기업 (average client ~150 employees 추정; "mid-market" 포지셔닝) |
| 수익 모델 | **Recurring** (92.3% FY25) = per-employee-per-month 구독 + 모듈당 add-on + interest on funds held for clients (페이롤 자금 플로트) |
| 고정비 vs 변동비 | 고정비 重 (R&D $282M·S&M·인프라). 변동비는 결제/클라우드 인프라 |
| 규모의 경제 | **작동 O** — recurring margin expansion + R&D/매출 ratio는 규모 커질수록 희석 여지 |

## 10. ⭐ 간이 3시나리오 Valuation (Fisher 트랙 → 10-B ROE×PBR, 2026-04-18 v2 소급)

**공통 가정**: r = 10% 고정, N = 10년
**기저 입력**: FY2025 ROE **18.4%**, ROIC 21.8%, PBR **~4.5x** (시총 $6.52B / equity ~$1.45B), 현재가 $117.85
**공식**: `r = (1+ROE) / PBR^(1/N) − 1` (터미널 PBR=1 수렴 가정)

> **주의 — 감속 관찰 중인 SaaS**: PCTY는 매출 CAGR 25.9%→9% 급감속 관찰기이고, SaaS의 float·buyback 구조로 ROE보다 ROIC가 적합할 수도. 본 간이 모델은 보수적 "터미널 PBR=1" 수렴 가정이어서 SaaS 해자 유지 시 과소평가 경향. Deep Stream C에서 EPS × PER 방식으로 재검증 필수.

| 시나리오 | 유지 ROE | PBR | 기대 CAGR | Fisher 판정 |

**확률가중 CAGR**: 0.25(7.6) + 0.55(3.3) + 0.20(−3.6) = **3.0%**

**확률 근거**: Bull 25% (ROE 감속 반전 필요 — Rippling 경쟁 환경에서 어려움), Base 55% (경영진 가이던스 +9% 지속 = Stalwart 이행 실현), Bear 20% (Rippling 침투 + AI commoditization).

### 10-B-4. Trigger 신호

- **Bull CAGR 7.6% < 10%** → Fisher 매수 가능 후보 **미해당**
- **Bull CAGR 7.6% > 4%** → 매수 금지 trigger **발동 안 함** (but 근접)
- **Base CAGR 3.3% < 4%** → 회색 하단, 실질 Stalwart 수준이면 매력 부족
- **판정 요약**: 현재 가격 $117.85는 Bull 시나리오에서도 r=10% hurdle 미달. Fisher 매수 가능 후보 기준 미충족 → **Keep 판정 타당** (Phase 11 확인)

### 10-B-5. Deep Stream C 연계 signal

- 10-B는 SaaS 해자 지속 시 **과소평가 경향** (terminal PBR=1 수렴 가정이 엄격). Deep Stream C의 EPS × PER 방식에서는 CAGR 2~4%p 상향 가능
- Deep 정밀화 핵심 축: (a) ROIC 27.9% 피크(FY24) 재현 가능성 (b) Airbase/AI ARPC 상승 실현도 (c) Rippling 침투 속도
- Keep → 1~2분기 감속 관찰 후 ROE 반등 증거 있으면 Bull 확률 상향 재검토. 없으면 탈락 후보.
